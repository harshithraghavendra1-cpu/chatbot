# Rule-Based Chatbot

This is a simple rule-based chatbot built using Python.

## Concepts Used:
- Control Flow
- Loops
- If-Else Statements
- Input Handling
- String Methods

## How to Run:
1. Open terminal
2. Run: python chatbot.py
3. Type messages and chat
4. Type 'exit' to stop

## Limitations:
- Only responds to predefined inputs
- Cannot understand complex sentences
